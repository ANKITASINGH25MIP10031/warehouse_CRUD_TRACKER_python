 **Problem Statement**  
Warehouse workers and inventory handlers often rely on manual registers or simple spreadsheets to keep track of stored items.  
These traditional methods can be slow, prone to mistakes, and difficult to update or maintain consistently.  
The **Warehouse Items CRUD Tracker** solves this problem by providing a simple, menu-driven Python console application that allows users to efficiently manage warehouse items.  
It supports adding, viewing, updating, and deleting inventory records in an easy and structured manner.



 **Scope of the Project**  
This project focuses on building a basic, console-based inventory management system using Python.  
The scope includes:

- Adding new warehouse items (Create)
- Viewing all stored items (Read)
- Updating existing item details (Update)
- Deleting items from the inventory (Delete)
- A simple and clear menu-driven interface
- Data stored in Python lists and dictionaries 



 **Target Users**
- Students learning CRUD operations  
- Beginners practicing Python programming  
- Small warehouse workers needing a simple offline tracker  
- Inventory interns or trainees  
- Anyone who needs a lightweight, easy-to-use item management tool  



 **High-Level Features**
- CRUD operations for warehouse items  
- Stores details like item name, quantity, location, and supplier  
- Simple command-line user interface  
- In-memory storage using lists and dictionaries  
- Easy navigation using menu options  
- Beginner-friendly structure and readable code
