# Warehouse Items CRUD Tracker

##  Project Overview
The **Warehouse Items CRUD Tracker** is a basic Python console application designed for beginners.  
It demonstrates all four CRUD operations:

- **C – Create** (Add new warehouse items)
- **R – Read** (View all items)
- **U – Update** (Modify item details)
- **D – Delete** (Remove items)

The program uses simple Python concepts such as:
- Lists  
- Dictionaries  
- Functions  
- Loops  
- Conditional statements  
- User input  


##  Features
###  Create  
Add a new warehouse item with:
- Item Name  
- Quantity  
- Storage Location  
- Supplier Name  

###  Read  
View all stored warehouse items in a clean list format.

###  Update  
Modify the quantity, location, and supplier of an existing item.

###  Delete  
Delete an item from the warehouse after confirmation.
